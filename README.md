# 📊 Случайные величины и распределения в Python

Интерактивный учебный ноутбук по теории вероятностей и статистике.

## 📚 Содержание
- Дискретные и непрерывные случайные величины
- Биномиальное, геометрическое, Пуассона, равномерное, экспоненциальное, нормальное распределения
- Практические задачи с решениями
- Визуализация графиков в Python

## 🛠 Используемые библиотеки
- NumPy, SciPy, Matplotlib, Seaborn

## 📖 Как использовать
Откройте ноутбук в Jupyter или Google Colab и изучайте материал.
